# 📱 Guide Rapide - Installation sur Mobile

## Pour les Organisateurs 🎯

### Étape 1 : Accéder à l'application
Ouvrez cette URL dans votre navigateur :
```
https://votre-domaine.com/static/index.html
```

### Étape 2 : Installer sur l'écran d'accueil

**Sur iPhone/iPad** :
1. Appuyez sur le bouton "Partager" (carré avec flèche vers le haut)
2. Faites défiler et choisissez "Sur l'écran d'accueil"
3. Appuyez sur "Ajouter"

**Sur Android** :
1. Appuyez sur le menu (⋮) en haut à droite
2. Choisissez "Installer l'application" ou "Ajouter à l'écran d'accueil"
3. Appuyez sur "Installer"

### Étape 3 : Utiliser l'application
- L'icône "PéléVTT 62" apparaît sur votre écran d'accueil
- Ouvrez-la comme n'importe quelle application
- Accès à la carte, configuration, suivi complet

## 💡 Astuces

### Économiser la batterie
- L'application utilise le GPS en continu
- Gardez un chargeur portable à portée de main
- Mettez l'écran en veille quand vous ne l'utilisez pas

### Vérifier la connexion
- Regardez l'indicateur de statut en haut
- 🟢 Vert = tout fonctionne
- 🟠 Orange = en attente (GPS, équipe...)
- 🔴 Rouge = problème de configuration

### En cas de problème
1. Vérifiez votre connexion Internet
2. Vérifiez que le GPS est activé
3. Fermez et rouvrez l'application
4. Si le problème persiste, contactez les organisateurs

## ❓ Questions fréquentes

**L'application consomme-t-elle des données mobiles ?**
Très peu. Une fois installée, la plupart des ressources sont en cache. Seules les données de position sont envoyées.

**Puis-je utiliser l'application hors ligne ?**
Oui, l'interface reste disponible. Les données seront synchronisées quand vous aurez à nouveau du réseau.

**Dois-je garder l'application ouverte en permanence ?**
Non. Déclarez juste l'état de votre équipe aux moments clés. L'application peut rester en arrière-plan.

**Puis-je avoir les deux applications ?**
Oui ! Elles sont indépendantes. Un organisateur peut avoir les deux.
